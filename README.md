# FewoFly Internal Tool

Internal operations tool for managing accommodation inquiries, assignments, and team workflows.

## Project Goal

This application is being developed as an internal business tool for FewoFly employees.  
Its purpose is to centralize and simplify inquiry intake, internal processing, and role-based access to operational data.

## Planned Core Features

- Secure employee login
- Role-based access control
- Inquiry creation and management
- Assignment and ownership tracking
- Filtering and status handling
- Light mode / dark mode
- Responsive internal dashboard

## Planned Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- TypeScript

### Database
- PostgreSQL
- Prisma ORM
